How it works

Client-side JS mirrors the Python backend: entropy math, regex-based pattern detection, curated breach samples, and leetspeak-aware name checks. The backend version adds deeper lists and richer heuristics.

Deploy & run the full analyzer

Use the Python backend for comprehensive checks (full blocklists, advanced patterns, API hooks).

  1. Clone: git clone https://github.com/ASantana0924/Password-Analyzer.git
  2. Install: pip install -r requirements.txt
  3. Run Flask: python flask_app.py
  4. Open: http://localhost:5000

Prefer static hosting? Point GitHub Pages to /docs on the gh-pages branch.
